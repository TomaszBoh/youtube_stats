DESCRIPTION

Simple project about creating ETL pipline which extracts the data from Youtube Data API and transforms it using PySpark and loads the data in AWS S3. Everything is orchesterd in Airflow.

TECH STACK:
- Docker
- Airflow
- PySpark
- AWS S3
- YouTube Data API V3
